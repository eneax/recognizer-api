## Recognizer-api

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. You have to add your own API key in the `controllers/image.js` file in order to connect to the Clarifai API.

You can grab a free Clarifai API key [here](https://www.clarifai.com/).
